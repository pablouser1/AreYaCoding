# Are Ya Coding, Son?
Code collaboration in real time

**IN DEVELOPMENT, CURRENTLY NOT USABLE**

## Usage
### Dependencies
#### Backend
```bash
go get .
```
#### Frontend
```bash
yarn install
```
### Building
```bash
make
```

Then, you just have to run:
```
./AreYaCoding
```
